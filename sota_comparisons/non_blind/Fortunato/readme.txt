% Part of Supplementary Materials
High-Quality Fast non-Blind Deconvolution Using Sparse Adaptive Priors

The MATLAB script "test_our_method.m" exemplifies the use of our technique (our_method_bifilter.m)
on three images: one synthetic and two natural ones. The \images folder contains the test images
and corresponding kernels. The "our_RF.m" script (called from our_method_bifilter.m) is a modified 
version of the Domain Transform RF filter by Gastal and Oliveira.

To perform the test type test_our_method_bifilter in MATLAB.
